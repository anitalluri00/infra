# DR Drill Checklist

## Quarterly Drill Checklist

- [ ] Confirm latest runbooks and contact lists
- [ ] Validate monitoring dashboards and alert channels
- [ ] Execute simulated primary outage
- [ ] Perform DR failover runbook end-to-end
- [ ] Validate business-critical transactions
- [ ] Measure achieved RTO and RPO
- [ ] Capture issues and remediation owners
- [ ] Run failback simulation
- [ ] Submit compliance evidence package
